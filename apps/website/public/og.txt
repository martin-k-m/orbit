OG image note
==============

This static export intentionally ships no binary OG/social image so the build
has zero missing-asset references. All visuals on the site are recreated in
CSS/SVG rather than bitmap images.

To add a real Open Graph image later:

1. Add `public/og.png` (recommended 1200x630).
2. In `app/layout.tsx`, extend the `openGraph` and `twitter` metadata:

     openGraph: {
       ...
       images: [{ url: "/og.png", width: 1200, height: 630 }],
     },
     twitter: {
       ...
       images: ["/og.png"],
     },

Because `metadataBase` is already set, Next.js will resolve the image to an
absolute URL automatically.
