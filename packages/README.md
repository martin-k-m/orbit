# packages/

Reserved for shared, reusable packages consumed by more than one app in the
Orbit monorepo — for example a future `@orbit/ui` design-system package shared
between the desktop frontend (`apps/desktop/ui`) and the website
(`apps/website`), or a shared TypeScript types package generated from the Rust
model.

Nothing lives here yet; it exists so the structure is ready to grow without
another reorganisation.
